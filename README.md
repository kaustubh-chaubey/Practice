# C# .NET Practice Repository

A collection of C# and .NET programming exercises designed to improve problem-solving skills, strengthen understanding of core language concepts, and prepare for technical interviews and real-world development.

## 📚 Purpose

This repository serves as a personal learning and practice space for:

- C# fundamentals
- Object-Oriented Programming (OOP)
- Data Structures and Algorithms
- LINQ
- Asynchronous Programming
- File Handling
- Design Patterns
- .NET Core/.NET Development
- Coding Challenges and Interview Preparation

---

## 🛠 Prerequisites

Before getting started, ensure you have:

- .NET SDK (latest stable version)
- Visual Studio, Visual Studio Code, or JetBrains Rider
- Basic knowledge of C# syntax

dotnet --version
